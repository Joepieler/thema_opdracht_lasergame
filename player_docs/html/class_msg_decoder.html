◆ main()

void MsgDecoder::main ( )
inlineoverride

This function contains the state machine.

The functions contains two states. In the idle state it waits for a pause and checks if the pause is valid. In the message state it checks if the following pause is valid and inserts a 1 or a 0 in the message respectively. If 16 valid pauses are detected the message gets sent with the messageDetected function and it returns to the idle state.
