void ShootControl::main ( )
inlineoverride

This function contains the state machine.

This function contains only one state called idle. The idle state waits for a shoot flag. When the shoot flag is set the laser turns on and the function setSendData is called. After 100ms the laser is turned off and the state starts waiting for the shoot_flag again.
